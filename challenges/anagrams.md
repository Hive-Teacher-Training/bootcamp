# Anagrams challenge

<p class="flow-text">

Use this <a href="/bootcamp/challenges/wordlist.txt">file of common words</a> to write a program
to find anagrams of words entered by a user.

</p>

e.g.
```

james-r@darkstar$ python3 anagrammer
Enter a word: race

Anagrams of 'race' are:

acer
care

```
